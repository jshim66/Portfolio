# Justin Shim Portfolio

React + Vite portfolio site built for simple Vercel deployment.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

## Deploy to Vercel

1. Push this `portfolio-site` folder to GitHub.
2. Import the repository into Vercel.
3. Vercel should auto-detect Vite and use:
   - Build command: `npm run build`
   - Output directory: `dist`

## Resume

If you want a live resume button, place your PDF at:

`public/Justin_Shim_SWE_Resume.pdf`

Then update the button href in `src/App.jsx`.
